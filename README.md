## Manual Installation
You can also construct chaus on your server.

### Mongo URL
Please set variable onto your environment or set `mongoURL` in `src/config.js`.

### API Host, Port
Please set API host, port in `src/config.js`.

### Starting application

```
npm run build
npm start
```
